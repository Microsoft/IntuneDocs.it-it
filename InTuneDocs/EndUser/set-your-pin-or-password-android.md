# Impostare il PIN o la password

Se si usa Intune per accedere ai dati aziendali o dell'istituto di istruzione, l'amministratore IT potrebbe richiedere l'impostazione di un PIN o di una password nel dispositivo Android. Se viene visualizzato un messaggio in cui si richiede di impostare un PIN o una password, attenersi alla procedura seguente.

1.  Toccare **Impostazioni** &gt; **Sicurezza** &gt; **Blocco schermo** &gt; **Password**.

2.  Scegliere e confermare la nuova password.


Serve ancora assistenza? Contattare l'amministratore IT. Per informazioni sul contatto vedere il [sito Web del portale aziendale](http://portal.manage.microsoft.com).
