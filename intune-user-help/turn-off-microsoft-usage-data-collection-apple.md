# <a name="how-to-turn-off-microsoft-data-collection"></a>Come disattivare la raccolta dati di Microsoft

Microsoft raccoglie automaticamente determinati dati per i suoi prodotti e servizi allo scopo di migliorare l'affidabilitÃ  e le prestazioni dei prodotti Microsoft, inclusa l'app Portale aziendale. Anche se questi dati sono resi anonimi, alcuni utenti potrebbero non acconsentire alla raccolta. Gli amministratori IT non hanno alcun controllo sulla raccolta di questi dati e non possono modificare la selezione di questa impostazione.

Per disattivare la raccolta dei dati di utilizzo nel dispositivo iOS, andare a **Impostazioni**, toccare **Portale aziendale** e quindi disattivare l'impostazione **Dati di utilizzo**. Per i dispositivi macOS, aprire il menu **Portale aziendale** > **Preferenze**, quindi deselezionare **Consenti a Microsoft di raccogliere i dati di utilizzo**.

Serve ancora assistenza? Contattare l'amministratore IT. Per informazioni sul contatto vedere il [sito Web del portale aziendale](http://portal.manage.microsoft.com).
