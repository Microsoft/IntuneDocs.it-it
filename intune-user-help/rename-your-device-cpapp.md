# <a name="rename-device-from-the-company-portal-app-for-windows"></a>Rinominare il dispositivo dall'app Portale aziendale per Windows
Rinominare un dispositivo per riconoscerlo più facilmente dagli altri dispositivi. L'app Portale aziendale per Windows consente di rinominare i dispositivi Windows, iOS e Android connessi. 

Per rinominare un dispositivo:
1. Aprire l'app Portale aziendale.
2. Nella sezione **Dispositivi personali** selezionare il dispositivo da rinominare.
3. Nell'angolo in alto a destra della pagina selezionare l'icona **Rinomina**. 
4. Digitare il nuovo nome e fare clic su **Rinomina**. L'aggiornamento del nome del dispositivo verrà immediatamente visualizzato nell'app. 

Serve ancora assistenza? Contattare l'amministratore IT. Per informazioni sul contatto vedere il [sito Web del portale aziendale](https://go.microsoft.com/fwlink/?linkid=2010980).
